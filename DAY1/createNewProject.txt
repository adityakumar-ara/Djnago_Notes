Run this command in teminnal 
command :- python -m django startproject projectName

👉 This creates a folder like:
myproject/
├── manage.py
└── myproject/
    ├── __init__.py
    ├── settings.py
    ├── urls.py
    ├── asgi.py
    └── wsgi.py


--------------------------------------------------------------------

Run the server 

first of all go to inside the folder through this command
command :- cd projectName

command :- python manage.py runserver
